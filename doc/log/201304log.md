ruby: duck typing, 会聒聒叫的就是鸭子
```ruby
# 鴨子
class Duck
  def quack
    puts "quack!"
  end
end

# 野鴨 (不用繼承)
class Mallard
  def quack
    puts "qwuaacck!! quak!"
  end
end

birds = [Duck.new, Mallard.new, Object.new]
# 迭代陣列，並呼叫方法 (無須擔心型別)
birds.each do |duck|
  duck.quack if duck.respond_to? :quack
end
```
> 语义是否有效是由对象的方法和属性决定的，而不是它继承自哪个类或者实现自哪个接口
